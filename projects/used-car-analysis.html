Used Car Market Analysis

R Data Analysis ggplot2 Statistical Analysis

Overview

A collaborative research project analyzing car performance data from the Corgis dataset to understand how various automotive specifications influence consumer preferences. The analysis provides insights to help consumers make informed decisions based on their needs.

Key Contributions

  • Data Preparation and Cleaning:
    • Imported and cleaned the dataset
    • Eliminated unnecessary columns
    • Verified data quality and structure
  • Feature Engineering:
    • Extracted numeric cylinder count from engine information
    • Converted data to appropriate formats for analysis
  • Statistical Analysis:
    • Computed average horsepower by cylinder count
    • Conducted correlation analyses
    • Created visualizations using ggplot2

Technologies Used

  • R Programming Language
  • ggplot2 for data visualization
  • stringr package for text processing
  • R Markdown for documentation

Impact

This analysis provides valuable insights into the relationship between vehicle specifications and performance, helping consumers make informed decisions when purchasing cars. The findings are documented in a comprehensive research paper with detailed visualizations and statistical analyses.

Team

Collaborative project with Jacob Spano, Joseph Cicero, Leo Morifuji, and Christian Cuevas.
